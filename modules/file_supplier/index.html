供应商账单拆分工具

使用说明

本工具用于拆分供应商账单文件，按供应商列拆分成独立的Excel和Word文件。所有处理过程均在浏览器中完成，不会上传数据到服务器。

  1. 上传包含供应商数据的Excel文件
  2. 系统会按照“供应商”列自动拆分文件
  3. 每个供应商生成一个独立的Excel文件（包含结算表和已发货表）和一个Word文档（服务费通知函）
  4. 可以批量下载所有文件（ZIP格式）或单独下载某个供应商的文件
